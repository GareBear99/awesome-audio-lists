# Looperman upload template

Looperman treats each loop as its own uploaded file. You upload per-loop; Looperman auto-builds your user pack page.

**Per-loop fields**:
- **Title**: `<short descriptive title>` (e.g. `Dark Piano Chord Loop — Cm 88 BPM`)
- **BPM**: exact integer
- **Key**: e.g., `C Minor`
- **Genre**: main genre
- **Sub-genre / tags**: up to 5
- **Instrument / sound type**: e.g., `Piano`, `Pad`, `Synth Lead`
- **Description** (~400 chars):
  ```
  <1–2 sentences describing loop mood + where it fits in a track>.
  Part of the free TizWildin <Pack Name> pack — https://github.com/GareBear99/<repo>.
  ```
- **License**: Royalty Free (Looperman default)
- **Preview**: 128kbps MP3 (Looperman auto-generates)
- **Download**: the WAV file itself

**Pack-level profile page update**:
Once 5+ loops from a pack are uploaded, link the GitHub repo from your Looperman bio/profile so the pack stays discoverable as a unit.

**Submission cadence**: 2–3 loops per day per pack; Looperman flags rapid mass-uploads.
